Производной функции y = f(x) в данной точке x0 называют предел отношения приращения функции к соотвтствующему приращению его аргумента при условии, что последнее стремится к нулю :
f'(x) = lim(x -> 0) = f(x0)/x
Дифференцирование - операция нахождения производной.

Таблица производных некоторых функций :
c (константа) => 0
x => 1
x^n => n*x^(n-1)
1/x => -1/x^2
e^x => e^x
ln(x) => 1/x
sin(x) => cos(x)
cos(x) => -sin(x)
tg(x) => 1/cos(x)^2
ctg(x) => -1/sin(x)^2

Основные правила дифференцирования :
1. (f(x) +- g(x))' = f'(x)  +- g'(x)
2. (f(x) * g(x))' = f'(x) * g(x) + f(x) * g'(x)
3. (f(x) / g(x))' = (f'(x) * g(x) - f(x) * g'(x)) / g(x)^2
4. f(g(x))' = f'(g(x)) * g'(x)

Физический смысл производной
Если материальная точка движется прямолинейно и её координата изменяется в зависимости от времени по закону x(t), то мгновенная скорость данной точки равна производной функции :
v(t) = x'(t)

Уравнение касательной для функции f(x) в точке a :
y = f(x) + f'(x) * (x - a)

Другие формулы :
S = f(b) - f(a) (нахождение площади функции при y >= 0 где f(x) - непрерывная функция, a и b - точки начала и конца отрезка)
Если функция g(x) параллельна функции f(x), то возможна следующая формула :
f'(x) = g'(x), из которой можно вынести абсциссу (x)
